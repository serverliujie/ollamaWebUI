# PHP Ollama Web UI 启动说明

## 快速启动

### Windows 系统

#### 方法 1: 双击启动（推荐）
1. 找到 `run_server.bat` 文件
2. 双击运行
3. 浏览器自动打开或手动访问 `http://localhost:8080/`

#### 方法 2: 使用启动器（带检查）
1. 双击运行 `start.bat`
2. 程序会自动检查 PHP 和 Ollama 状态
3. 启动成功后访问 `http://localhost:8080/`

#### 方法 3: PowerShell 启动
1. 右键点击 `start.ps1`，选择"用 PowerShell 运行"
2. 或者在 PowerShell 中执行：
   ```powershell
   .\start.ps1
   ```

### Linux/Mac 系统

#### 方法 1: 使用启动脚本
```bash
# 给脚本添加执行权限
chmod +x run_server.sh

# 运行启动脚本
./run_server.sh
```

#### 方法 2: 直接运行 PHP 命令
```bash
php -S 0.0.0.0:8080 index.php
```

## 启动文件说明

| 文件名 | 适用系统 | 功能说明 |
|--------|----------|----------|
| `run_server.bat` | Windows | 简单启动，双击运行 |
| `start.bat` | Windows | 带检查的启动器 |
| `start.ps1` | Windows | PowerShell 启动器 |
| `run_server.sh` | Linux/Mac | Shell 启动脚本 |
| `run.sh` | Linux/Mac | 原始启动脚本 |

## 启动后的操作

1. **访问 Web 界面**
   - 打开浏览器访问 `http://localhost:8080/`
   - 或点击启动窗口中的链接（如果支持）

2. **选择模型**
   - 在页面顶部的下拉菜单中选择一个模型
   - 如果只有一个模型，会自动选择

3. **选择对话模式**
   - **单次对话**：每次对话独立，不保留历史
   - **连续对话**：保留对话历史，支持多轮对话

4. **开始对话**
   - 在输入框中输入问题
   - 按 Enter 或点击"发送"按钮
   - 等待 AI 回复

## 常见问题

### 1. 启动失败：端口被占用
如果 8080 端口被占用，可以修改启动命令：
```bash
php -S 0.0.0.0:8081 index.php
```
然后访问 `http://localhost:8081/`

### 2. PHP 未找到
- 确保 PHP 已安装并添加到系统 PATH
- Windows: 在命令行输入 `php --version` 检查
- Linux/Mac: 在终端输入 `php --version` 检查

### 3. Ollama 未运行
- 确保 Ollama 服务器正在运行
- 在命令行输入 `ollama serve` 启动 Ollama
- 或使用 `start.bat`，它会自动尝试启动 Ollama

### 4. 无法访问 Web 界面
- 检查防火墙设置，允许 8080 端口
- 确保 PHP 服务器已成功启动
- 检查浏览器是否正常工作

### 5. 中文显示乱码
- 确保系统区域设置支持 UTF-8
- 使用现代浏览器（Chrome、Edge、Firefox）

## 停止服务器

- **Windows**: 在启动窗口中按 `Ctrl+C`，或关闭窗口
- **Linux/Mac**: 在终端中按 `Ctrl+C`

## 开发环境启动

如果你想在开发环境中启动（带调试信息）：

### Windows
```batch
php -d display_errors=1 -d error_reporting=E_ALL -S 0.0.0.0:8080 index.php
```

### Linux/Mac
```bash
php -d display_errors=1 -d error_reporting=E_ALL -S 0.0.0.0:8080 index.php
```

## 测试是否启动成功

### 方法 1: 浏览器测试
打开浏览器访问 `http://localhost:8080/`，应该能看到 AI 聊天界面。

### 方法 2: 命令行测试
```bash
# 测试 API 端点
curl http://localhost:8080/api/tags

# 测试生成端点
curl -X POST http://localhost:8080/api/generate \
  -H "Content-Type: application/json" \
  -d '{"model":"qwen3:0.6b","prompt":"你好","stream":false}'
```

## 技术要求

- **PHP**: 7.4 或更高版本
- **cURL**: PHP cURL 扩展
- **mbstring**: 推荐安装，用于中文字符处理
- **Ollama**: 正在运行的 Ollama 服务器

## 获取帮助

如果启动过程中遇到问题：
1. 检查 `test.php` 脚本的输出
2. 查看 PHP 错误日志
3. 确保 Ollama 服务器正常运行
4. 检查网络连接和防火墙设置
